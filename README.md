# 📘 Dashboard Rapor Otomatis SD Negeri Wonokerto

Sistem ini membantu guru menghasilkan deskripsi **Kokurikuler, Ekstrakurikuler (Siaga & Penggalang), dan Catatan Wali Kelas** secara otomatis, lalu mengekspor hasilnya ke file Excel/CSV.

---

## 🔐 Login
- Buka `index.html` di browser.
- Masukkan sandi: **saefulsdnwonokerto**.
- Jika benar, akan masuk ke dashboard.

---

## 🏠 Struktur Dashboard
- **Upload Data Siswa** → input No, Nama, NIS.
- **Kokurikuler** → pilih skor tiap aspek (1–4), klik *Generate Kokurikuler*.
- **Ekstrakurikuler Siaga** → pilih skor tiap aspek, klik *Generate Siaga*.
- **Ekstrakurikuler Penggalang** → pilih skor tiap aspek, klik *Generate Penggalang*.
- **Catatan Wali Kelas** → input ranking (1–30), klik *Generate Catatan*.
- **Export Hasil** → klik *Download Excel* untuk menyimpan data.
